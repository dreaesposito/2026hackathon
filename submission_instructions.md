# Submission Instructions

1. **Fork this repository** to your own GitHub account (or your team’s).

2. **Make the fork public** so we can access it.

3. **Submit via the Google Form** (link in the main README). In your submission, include the link to your public fork.

4. **Only the most recent commit on `master` before the deadline will be considered.** Make sure everything you want graded is in that commit and on the `master` branch.
